## Project Overview
wi-unity-game is an FPS (First Person Shooter) game prototype with roguelike elements developed using Unity Engine.  
This project was created as part of the **Advanced Game Development** course to demonstrate the ability to analyze, design, and implement advanced game mechanics and scripting.

## Core Gameplay Loop
1. Player spawns in the level
2. Enemies spawn randomly
3. Player explores and engages enemies
4. Combat continues until player is defeated
5. Game over and restart option available

---

## Key Features
- FPS Player Controller (movement & camera)
- Enemy AI (detection, chase, attack)
- Roguelike random enemy spawning
- Shooting & damage system
- Health and score UI
- Game state management (Menu, Play, Game Over)
